Useage:

To use these, simply create a shortcut on your bookmarks bar in your browser, then right click it and click "edit". Next copy the code from the textarea that you wish to use, and paste it into the "URL" box of the shortcut, then save it. Done!

updating page:

This shortcut will navigate to the updating version of the current page, if one exists.

! updating page:

This shortcut will navigate from the updating page to the non-updating page.

view list:

This shortcut will navigate to the list structure behind the current page.

Remove params:

This shortcut will reload the page without url parameters.

Add test param:

This shortcut will reload the page with a test parameter.

Return link to grid:

This shortcut will produce a popup with the UNC path to the grid on the current page.

Return link to grid (sans file name):

This shortcut will produce a popup with the UNC path to the grid on the current page.

Get the UNC path to the current page:

This shortcut will try to produce a popup with the UNC path to the current page. This may not always work, it mostly just works for pages that are based on a sharepoint site.
